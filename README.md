# 帽制片 · 选题雷达（测试版 v0.1）

面向制片人的「选书雷达」——每周从豆瓣阅读，在大买家之前，挑出还没卖、又有改编潜力的早期作品。

这是「帽制片」工作台**策划板块**的第一个可用版本。制作（剧本拆解）、发行板块在路线图里。

---

## 🚀 怎么运行

### 方式一：双击启动（推荐）
双击 **`启动.command`**，会自动打开本地服务器并在浏览器里打开 App。
（首次双击若提示安全限制，在「系统设置 → 隐私与安全性」里点「仍要打开」即可。）

### 方式二：直接打开
双击 `app/index.html`，用浏览器打开。
（注：直接打开时，"收藏 / 点赞"在部分浏览器可能不保存；方式一更稳。）

---

## 🧭 这个版本能做什么

- **选品原则：好故事 > 题材**。每部两个分——**故事力**（智能体的文学判断）+ **题材匹配**，综合 = 故事力×0.6 + 题材×0.4。
- **本周精选** + 备选库（来源：豆瓣阅读 + 晋江），每部带完整改编分析报告。备选含电竞 / 花滑等竞技作（耽美，改编在剧本层转换人物关系即可）。
- **口味偏好自选**：左侧勾选你喜欢 / 不感冒的题材、体量、状态 → 自动生成偏好总结（面向多制片人，不写死）
- **筛选 / 搜索**：按题材、**来源**、状态、关键词
- **收藏 + 点赞 / 拍掉**：反馈会用来校准（未来回流到智能体记忆）
- 点任意卡片 → 看完整报告（**好故事判断**、梗概、人物、改编亮点 / 难点、对标、版权线索、总评）

## ⚠️ 关于数据真实性（重要）

本周内容是**真实的豆瓣阅读作品**：

| 真实（来自平台页面 / 搜索索引） | 智能体判断（主观分析） |
|---|---|
| 书名、作者、链接、字数、状态、梗概 | 故事力、题材匹配、改编亮点 / 难点、对标作品、综合指数、总评 |

- 本周精选（《汴京小面馆》《欢迎回家》《攻玉》）**链接均已核实**。
- 备选库里标「链接待核实」「故事待评估」的，是真实书名 / 作者，但直达链接或故事质量还需进一步确认。

- **版权状态**只是线索，不保证准——务必正式核实后再决策。
